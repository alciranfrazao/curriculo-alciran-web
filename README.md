## 🎯 Sobre o Projeto

Este site serve como currículo interativo e portfólio, mostrando minhas experiências, formação e habilidades técnicas de forma visual e organizada.

**Tecnologias utilizadas:**
- HTML5 semântico
- CSS3 com variáveis, Flexbox e animações
- JavaScript puro (sem frameworks) para interatividade
- Google Fonts
- Design responsivo (funciona em celular e desktop)

**Funcionalidades:**
- Animação de digitação na tela inicial
- Menu hambúrguer para celular
- Barras de habilidades animadas ao fazer scroll
- Cards com animação de entrada
- Navegação entre páginas com navbar fixa
