## 🎯 Sobre o Projeto

Este site serve como currículo interativo e portfólio, mostrando minhas experiências, formação e habilidades técnicas de forma visual e organizada.

**Tecnologias utilizadas:**
- HTML5 semântico
- CSS3 com variáveis, Flexbox e animações
- JavaScript puro (sem frameworks) para interatividade
- Google Fonts
- Design responsivo (funciona em celular e desktop)

**Funcionalidades:**
- Animação de digitação na tela inicial
- Menu hambúrguer para celular
- Barras de habilidades animadas ao fazer scroll
- Cards com animação de entrada
- Navegação entre páginas com navbar fixa

## 📤 Como atualizar no GitHub (passo a passo simples)

> Faça isso sempre que modificar algum arquivo do projeto.

### Pré-requisito: Git instalado no computador
Se não tiver, baixe em: https://git-scm.com/downloads

### Na primeira vez (configurar o Git):
```bash
git config --global user.name "Alciran Frazão"
git config --global user.email "alciranfrazao@gmail.com"
```

### Para enviar atualizações:

**1. Abra o terminal na pasta do projeto**
   - Windows: botão direito na pasta → "Git Bash Here" ou "Abrir no Terminal"

**2. Verifique o que mudou:**
```bash
git status
```
Vai mostrar em vermelho os arquivos que você alterou.

**3. Adicione os arquivos modificados:**
```bash
git add .
```
O ponto (.) significa "adicionar tudo". Você pode colocar só o nome do arquivo se quiser.

**4. Crie um commit com uma mensagem explicando o que mudou:**
```bash
git commit -m "Atualiza seção de experiências com Fecomércio RN"
```
A mensagem fica salva no histórico. Seja descritivo (ex: "Adiciona animação nas barras de skill").

**5. Envie para o GitHub:**
```bash
git push origin main
```
(Se o branch principal for `master`, use `git push origin master`)

**6. Aguarde 1-2 minutos e o GitHub Pages atualiza automaticamente!**
